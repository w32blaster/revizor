Social Coding

Open Source solution for code review

Revizor — is a code review tool and a repository browser. It allows you to build a social coding atmosphere around your code base. Keep track on what is going in your repositories.

Why Revizor?

Choose right tool for your repository

Open source

Revizor is free and open source application, that is hosted on GitHub and written on Groovy on Grails.

Simple to install

Revizor can be executed using only one command java -jar, because it is distributed as self-contained archive (.jar) or deployed to your favorite container as .war file.

Smart commits

You can send simple commands to Revizor in a commit message in your favourite CVS. Check the docs.

Integrations

Revizor can be easily integrated with issue trackers or chats. Check the list of supported services.

Mission

Code review — it is not only about inspection and controlling. It is more about communication within a code. The development should be socialized, allowing to share the ownerhip of the whole product across the team. Revizor is aimed to give a free tool to connect programmers in a code context, issue trackers and CVS system.

Easy to get. Easy to install.

A development tool should not be complicated. Revizor is aimed to be easy to start working. There is only one requirement - Java Runtime Environment. If you have JRE installed on your server, you are already can use Revizor. Next step you need to add your repositories and you are ready to go!
